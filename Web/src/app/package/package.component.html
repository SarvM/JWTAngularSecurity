After successful authentication, based on claims respective block will be displayed. We have 3 claims to demonstrate the authorization. 1. Admin, 2. Manager, 3. Employee

Menu, Block, and Checkout Button are for demonstration purpose, to showcase how it works based on claims. It's enabled/disabled based on claims

Free

  • * No Authorization Required
  • * Displays for all claims

Admin

  • * Authorization Required
  • * Only for Admin Claim

Manager

  • * Authorization Required
  • * Admin and Manager Claim

Employee

  • * Authorization Required
  • * Admin, Manager and Employee can Access
